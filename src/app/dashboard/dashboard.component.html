API

We created Azavea's Climate API in order to give you easy access to the NASA NEX-GDDP data without the hassle of having to download and wrangle a NetCDF dataset. With each query, you can choose from over 1,000 U.S. locations, either of the two most commonly used RCPs, 22 indicators of temperature and precipitation, and anytime 1950-2100.

To get started with the API, please review the documentation.
